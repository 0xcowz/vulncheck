Check des vulnérabilités existantes sur un poste par une API publique.

Script in two steps :
- 1 > csv_creator.ps1 > Will create the .csv file that contain all the vuln's on the PC using CISA KEV
- 2 > csv_checker.html > Web page that will show clearly all the vuln's that has been found on the PC (you'll need to import manually the .csv file due to Edge protection)
